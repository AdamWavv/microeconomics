# Project Overview

The objective of this project is to specify, estimate, and verify a microeconometric model using the STATA package. The analysis is based on a selected variable from the EU-SILC dataset, which contains comprehensive microdata on income and living conditions in the European Union.

# Data Source

Data for this project can be obtained from the official Eurostat website:

[EU-SILC Microdata](https://ec.europa.eu/eurostat/web/microdata/public-microdata/statistics-on-income-and-living-conditions)

Additional information about the dataset, including variable descriptions and data structure, can be found in the following document:

[EU-SILC User Guide](https://ec.europa.eu/eurostat/documents/203647/16195750/2021_Doc65_EUSILC_User_Guide.pdf)


# Course Program

The course covers econometric methods focused on microdata. The subject matter is of significant practical importance and highly relevant to economic sciences (notably, the 2000 Nobel Prize in Economics awarded to J. Heckman and D. McFadden). The syllabus includes topics related to modeling qualitative and limited dependent variables.

# Course Objective

The objective of the course is to develop the ability to specify, estimate, and validate econometric models based on microdata.

# Project Guidelines

The project involves the specification, estimation (the modeling must be conducted using the STATA software package), and validation of a microeconometric model for a selected variable, based on a sufficiently large set of microdata. Techniques discussed during the course must be applied. The data may come from various sources. The only thematic restriction: projects analyzing determinants of happiness or similar topics will not be accepted.
